# Image Guidelines

## Folder Structure

- `/hero` - Hero section background images
- `/services` - Service-related images (server work, coding, etc.)
- `/portfolio` - Project screenshots and work samples
- `/about` - Team and office photos

## Recommended Images to Upload

### For Hero Section
- `hero-server-room.jpg` - Dell servers in HPC center
- `hero-debugging.jpg` - Server debugging work

### For Services Section
- `service-hardware.jpg` - Dell server motherboard installation
- `service-debugging.jpg` - Server troubleshooting
- `service-coding.jpg` - Development work
- `service-cloud.jpg` - Cloud infrastructure setup

### For Portfolio/About
- `project-hpc-center.jpg` - HPC center installation
- `project-server-upgrade.jpg` - Server upgrade work
- `team-working.jpg` - Team collaboration

## Image Specifications

- Format: JPG or PNG
- Max size: 2MB per image
- Recommended dimensions:
  - Hero: 1920x1080px
  - Services: 800x600px
  - Portfolio: 1200x800px

## How to Upload

1. Place your photos in the appropriate folders
2. Name them descriptively (e.g., `dell-server-installation.jpg`)
3. Update the component files to reference your images
